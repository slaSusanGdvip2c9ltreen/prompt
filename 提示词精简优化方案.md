# Sora提示词精简优化方案

## 当前状况
- **总行数：** 706行
- **问题：** 内容过长，重复说明多，对于没有上下文的AI可能信息过载

## 优化目标
1. 减少至少30%的长度（目标：~500行）
2. 保持逻辑完整性
3. 确保无上下文AI也能完全理解
4. 删除冗余，保留精华

---

## 优化策略分析

### 策略1：合并重复的概念说明

**发现的重复：**

1. **"风格世界观"说明重复**
   - 最高法则二（第34-56行）详细列举了5种分类
   - 工作流程中又重复引用
   - **优化：** 保留法则二的详细说明，其他地方只引用

2. **"音频类型"说明重复**
   - 第五步详细说明了4种音频类型（第171-174行）
   - 提示词结构中又列举了4种模板（第268-284行）
   - **优化：** 合并为一处完整说明

3. **"真实性"概念重复强调**
   - 真实性层级判断（第71-102行）
   - 用户认知预设中又强调真实性（第136-140行）
   - **优化：** 真实性层级只说一次，预设部分简化引用

### 策略2：精简详细的子维度说明

**过于详细的部分：**

1. **维度4（视觉层分析）有4个子维度，每个都有详细说明**
   - 色彩对比（3个问题）
   - 动态趋势（3个问题）
   - 新奇元素（2个问题）
   - 空间景深（2个问题）
   - **优化：** 改为简洁的检查清单格式，每个维度1-2句话

2. **维度5（情绪层分析）有3个子维度**
   - 核心情绪触发机制（2个问题）
   - 拟人化特征（2个问题）
   - 情绪高潮点（2个问题）
   - **优化：** 同样改为检查清单

3. **"开头3秒的关键性"说明过长（第131-145行）**
   - 动态冲击感（4个例子）
   - 视觉钩子（2个例子）
   - 猎奇元素（2个例子）
   - 悬念设置（2个例子）
   - **优化：** 保留概念，每个只保留1个最佳例子

### 策略3：删除过度说明的示例

**示例过多：**

1. **创新策略工具箱（第433-460行）**
   - 6种策略，每种都有详细说明和多个示例
   - **优化：** 每种策略只保留核心原则和1个最佳示例

2. **Sora友好提示词示例库（第639-667行）**
   - 示例1有3个子示例（1、1-B、1-C）
   - 示例2、示例3
   - **优化：** 每个音频类型只保留1个最精简的示例

3. **空间关系示例过长（第413-417行）**
   - 正确示例用了100多字
   - **优化：** 缩短到50字以内

### 策略4：简化多层嵌套的结构

**嵌套过深：**

1. **身份符号因子（第201-251行）**
   - 职业身份标志（5种，每种有详细描述）
   - 社会角色标志（4种）
   - 文化符号标志（3种）
   - 强制规则、错误示例、案例对比、创新约束
   - **优化：** 简化为表格形式，每类只列举核心项

2. **社会性情绪说明（第149-198行）**
   - A类5种情绪，每种有详细说明（触发元素、核心价值、案例）
   - B类8种情绪
   - 判断优先级、强制检查清单
   - **优化：** 改为简洁的两列表格

### 策略5：删除冗余的强调和提醒

**过度强调：**

1. **多处使用【强制要求】【关键】【重要】等标记**
   - **优化：** 只在真正关键的地方使用，其他改为正常表述

2. **"特别提醒"部分（第674-706行）重复了前文内容**
   - **优化：** 只保留最核心的提醒，其他删除

3. **多处重复"不可违背"、"绝对禁止"等表述**
   - **优化：** 统一为简洁的约束列表

## 具体优化方案

### 优化1：紧凑的核心概念（替代第12-24行）

**原版（13行）：**
```
核心概念澄清 (Concept Clarification)
在开始工作前，请明确以下核心概念的定义和关系：
【爆款逻辑】 = 视频为什么爆的完整原因（可能包含多个因素）
例如："反差萌 + 情绪共鸣 + 视觉冲击"
【核心情绪价值】 = ...
【爆款因子】 = ...
【风格世界观】 = ...
【音频基因】 = ...
```

**精简版（6行）：**
```
核心概念速查
【爆款逻辑】= 视频为什么爆（如：反差萌+情绪共鸣+视觉冲击）
【核心情绪价值】= 触发的情感（如：保护欲、职业敬意）
【爆款因子】= 承载情绪的元素（身份符号>动作>场景）
【风格世界观】= 视频风格（模拟现实/荒诞夸张/写实纪录/电影叙事/魔幻现实）
【音频基因】= 音频构成（口播/BGM/音效）| 【真实性层级】= 用户相信程度（1-3级）
```

### 优化2：表格化的情绪分类（替代第147-198行）

**原版（51行）：** 详细列举A类5种、B类8种情绪，每种都有说明

**精简版（15行）：**
```
核心情绪价值（优先级：A类 > B类）

| A类-社会性情绪（传播力最强） | 触发元素 | B类-个人情绪 | 触发元素 |
|---|---|---|---|
| 职业敬意 | 军装/白大褂/警服/消防服/教师 | 幽默搞笑 | 意外/反转 |
| 爱国情绪 | 国旗/军装/国歌/民族符号 | 可爱反差萌 | 角色错位 |
| 公平正义 | 惩恶扬善/弱者反抗 | 视觉奇观 | 震撼画面 |
| 社会共情 | 老人/残疾人/贫困/弱势 | 保护欲/温情 | 脆弱/纯真 |
| 社会角色认同 | 母亲/父亲/英雄标志 | 惊吓/好奇 | 危险/违常识 |

判断规则：
✓ 先检查画面中是否有职业制服、国旗、社会角色标志
✓ 如有 → A类是核心（占70-90%），B类是次要
✓ 如无 → B类是核心
✓ 案例：军人回家 = 职业敬意80% + 家庭温情20% → 创新必须强化军装，而非温情
```

### 优化3：表格化的爆款因子（替代第199-251行）

**原版（52行）：** 详细说明三层、每层有详细列举和示例

**精简版（20行）：**
```
爆款因子三层识别（优先级：第一层 > 第二层 > 第三层）

| 层级 | 因子类型 | 核心元素 | 价值占比 |
|---|---|---|---|
| 第一层 | 身份符号 | 职业标志（军装/白大褂/警服/消防服/教师标志）<br>社会角色（母亲/父亲/老人/残疾标志）<br>文化符号（国旗/宗教符号/仪式） | 60-90% |
| 第二层 | 核心动作 | 承载情绪的具体动作、表情、行为 | 10-30% |
| 第三层 | 关键场景 | 特定环境、道具、氛围 | 5-10% |

强制规则：
✓ 如画面中存在身份符号 → 必须列为第一爆款因子
✓ 创新时：第一层必须强化，第二层可强化，禁止弱化第一层
✓ 案例对比：
  - ❌ 军人回家的爆款因子是"婴儿爬行"（错误：忽略身份符号）
  - ✅ 爆款因子是"父亲的军装80% + 婴儿爬行20%"（正确：身份符号优先）
```

### 优化4：简化空间关系说明（替代第383-432行）

**原版（50行）：** 5个步骤，每个步骤都有详细说明

**精简版（25行）：**
```
空间关系强制分析（内部完成，体现在输出中）

必须在生成提示词前完成：
1. 建立坐标：镜头位置 + 所有角色位置 + 关键物体位置
2. 规划路径：每个移动的起点、方向、距离、速度、终点
3. 检查逻辑：距离合理性、方向一致性、动作可行性

输出标准（必须能画出平面图）：
✅ 正确："婴儿从画面中央（距镜头2米）面向镜头爬行1.5米到镜头前（0.5米处）。母亲在镜头后拍摄。婴儿转身180度面向画面右后方的父亲（距离5米），父亲已从门口（画面深处）走到游戏垫旁（画面右侧中景）单膝跪下。婴儿全速爬行4-5米到达父亲身边。"
❌ 错误："婴儿爬向母亲，然后爬向父亲。" ← 位置不明

强制要求：
• 每个角色必须标明初始位置和距离
• 每个移动必须包含起点、方向、距离、终点
• 读者能根据描述画出场景平面图 = 合格
• 做不到 = 必须重写
```

### 优化5：精简音频指令（替代第266-284行）

**原版（19行）：** 4种类型，每种都有格式和示例

**精简版（10行）：**
```
音频指令模板（根据第五步分析结果选择）

1. 纯环境音："视频只包含自然的环境音效，绝对没有任何背景音乐或BGM。"
2. BGM配乐型："视频配有[类型]风格的背景音乐，[情绪]的旋律，[节奏]，[卡点/烘托]。"
3. 口播解说型（口播必须英文）："视频包含[身份]的口播，内容为：\"[英文翻译]\"，语调[风格]。[如有BGM/环境音继续描述]"
4. 综合型："视频包含[口播-英文]，配有[BGM]，以及[关键音效]。"

口播翻译规则：原语言→英文，保持原意和语言风格（幽默保留笑点，解说保持专业），用英文双引号括起。
```

### 优化6：删除重复的提醒（替代第674-706行）

**原版（32行）：** 特别提醒部分重复了大量前文内容

**精简版（12行）：**
```
执行检查清单
✓ 真实性：3级视频禁止电影化，保持UGC质感
✓ 身份符号：如有职业标志/国旗等，必须列为第一爆款因子并强化
✓ 社会情绪：如有职业敬意/爱国等，优先于个人情绪
✓ 空间清晰：每个移动必须能画出平面图
✓ 音频完整：口播翻译为英文，BGM详细描述
✓ 结构复刻：分镜数量、时长比例严格一致
✓ 字数匹配：每秒30-40字，根据分镜时长调整
```

2. ✅ 五维解构的框架

- 部分表格化
- 简化重复内容
