# 提示词输出问题深度分析与底层解决方案

## 问题1：对爆款核心理解不准确

### 案例分析：军人回家视频

**AI的错误理解：**
```
核心情绪价值：强烈的"家庭温情"与"纯真喜悦"共鸣
爆款因子：婴儿的"狂喜加速爬"动作
```

**真正的爆款核心：**
- ❌ 不是"家庭温情"（这是任何家庭视频都有的，没有独特价值）
- ✅ 是"用户误以为真实视频 + 军人身份"触发的**社会性情绪**：
  - 对军人的敬畏和尊重
  - 对军人家庭牺牲的认同
  - "铁汉柔情"的身份反差
  - 爱国情绪的共鸣

**AI的错误设计方向：**
三个方案都在强化"家庭温情"，完全忽略了**军人身份**才是核心价值：
- 方案1：只强化婴儿的情绪变化
- 方案2：引入泰迪熊，但仍然是家庭道具，没有强化军人身份
- 方案3：双胞胎，只是数量放大，核心价值没有变

**正确的设计方向应该是：**
- 强化军装的视觉呈现（徽章、装备细节）
- 增加军人职业特征的细节（敬礼、站姿、军用背包）
- 保持UGC的"真实性"，让用户相信这是真实记录
- 环境中增加军人元素（军旗、军人照片等）

## 问题2：动作和位置描述不够清楚

### 案例分析：两个案例都存在的问题

**具体问题示例：**

1. **空间关系模糊：**
```
❌ "婴儿爬向镜头方向的母亲"
问题：母亲在哪里？镜头在哪里？婴儿从哪里开始爬？距离多远？
```

2. **动作逻辑不清：**
```
❌ "父亲走到游戏垫旁并单膝跪下"
问题：父亲原来在哪里？走了多远？什么时候开始走？什么时候跪下？
```

3. **位置变化混乱：**
```
❌ "婴儿转身，目光锁定在父亲身上"
问题：婴儿之前面向哪里？转了多少度？父亲在婴儿的什么方向？
```

## 根本原因分析

### 原因1：缺少"真实性与社会价值"的分析维度

**现有问题：**
- "风格世界观"只描述了视觉风格（模拟现实、荒诞夸张等）
- 但没有区分"看起来像真的"和"用户真的相信是真的"
- 没有分析"真实性"如何放大情绪

**关键洞察：**
- UGC风格的军人视频 vs 电影风格的军人视频，爆款逻辑完全不同
- 前者触发"社会敬意"，后者只是"故事感动"
- 真实性是情绪放大器

### 原因2：缺少"社会文化价值"的分析维度

**现有问题：**
- "核心情绪价值"只列举了基础情绪（幽默、可爱、温情、保护欲）
- 完全忽略了社会性情绪（敬畏、尊重、爱国、职业认同、社会角色认同）

**关键洞察：**
- 军人、医生、消防员、老师等职业身份本身就是强大的情绪触发器
- 职业制服、身份标志是比动作更重要的"爆款因子"
- 社会性情绪比个人情绪传播力更强

### 原因3："爆款因子"识别过于表面化

**现有问题：**
- 只识别"动作"层面（婴儿爬、狗叼）
- 忽略"身份符号"层面（军装、徽章、职业特征）

**案例对比：**
| 表面因子 | 深层因子 |
|---------|---------|
| 婴儿开心地爬 | 婴儿爬向**穿军装**的父亲 |
| 家庭团聚温情 | **军人**家庭的团聚 |
| 父亲拥抱孩子 | **刚执行任务归来的军人**拥抱孩子 |

**正确的识别方式：**
- 身份符号 > 动作
- 社会意义 > 个人情绪
- 真实性 > 美观度

### 原因4：缺少"用户心理预设"分析

**现有问题：**
- 没有分析用户看到视频的第一反应是什么
- 没有分析视频如何利用或打破用户的预设认知

**关键洞察：**
- 军人视频：用户预设"这是真实记录" → 触发敬意
- 如果用户识破"这是摆拍" → 情绪价值归零
- 保持真实性是这类视频的生命线

### 原因5：缺少"空间关系"的强制分析

**现有问题：**
- 提示词要求描述"详细"，但没有要求描述"空间关系"
- AI倾向于描述情绪和氛围，忽略物理位置

**导致的后果：**
- 婴儿突然出现在不可能到达的位置
- 父亲的移动路径不清晰
- 镜头视角和角色位置关系混乱

## 底层解决方案

### 解决方案1：增加"真实性层级"判断

在"风格世界观"之后，增加"真实性评估"维度：

```markdown
【真实性层级判断】
判断视频属于以下哪一层级：
1. 明显虚构（用户知道是拍摄/表演）
   - 影响：情绪基于"故事好看"
   - 例子：微电影、广告片

2. 半真实（用户不确定真假）
   - 影响：情绪被"可能是真的"放大
   - 例子：高质量的恶搞视频

3. 完全真实（用户相信100%是真实记录）
   - 影响：情绪被"这真的发生了"指数级放大
   - 例子：UGC监控录像、手机随手拍

【关键】：对于"完全真实"层级的视频，任何破坏真实性的创新都会摧毁爆款价值。
```

### 解决方案2：增加"社会文化价值"维度

在"核心情绪价值"中，增加社会性情绪类别：

```markdown
维度2（扩展版）：识别核心情绪价值
分为两类：

【个人情绪类】（原有）
- 幽默搞笑、可爱反差萌、视觉奇观、保护欲、温情等

【社会性情绪类】（新增，优先级更高）
- 职业敬意：对特定职业的尊重（军人、医生、消防员、教师、警察）
- 社会角色认同：对某种社会角色的认同（母亲、父亲、英雄、弱者、老人）
- 爱国情绪：国旗、军装、国歌等元素触发
- 公平正义：惩恶扬善、弱者反抗
- 社会共情：对弱势群体的同情

【判断优先级】：
如果视频同时包含个人情绪和社会性情绪，社会性情绪是核心价值。
例如：军人回家 = 社会敬意（核心）+ 家庭温情（次要）
```

### 解决方案3：增加"身份符号"识别

在"爆款因子"识别中，强制要求识别身份标志：

```markdown
维度3（扩展版）：锁定"爆款因子"

分为三个层次（从高到低）：

【第一层：身份符号因子】（最重要，优先识别）
- 职业标志：军装、医生白大褂、警服、消防服、工作证
- 身份标志：结婚戒指、怀孕、残疾、年龄特征
- 文化符号：国旗、宗教符号、特定仪式

【第二层：动作因子】
- 承载情绪的具体动作

【第三层：场景因子】
- 特定的环境和道具

【强制规则】：
如果画面中存在身份符号，必须将其作为第一爆款因子。
例如：军人回家视频
- ✅ 爆款因子：父亲的军装（身份符号）
- ❌ 爆款因子：婴儿的爬行（动作）
```

### 解决方案4：增加"用户认知预设"分析

在"爆款逻辑分析"前，增加用户视角分析：

```markdown
第二步（新增）：用户认知预设分析

【强制思考】在分析爆款逻辑前，先回答：

1. 用户第一眼会认为这是什么类型的视频？
   - 真实记录？表演？新闻？恶搞？

2. 用户的预设认知是什么？
   - 他们期待看到什么？
   - 他们会相信视频内容吗？

3. 视频的爆款价值依赖于用户的哪种预设？
   - 如果用户识破"是假的"，爆款价值会如何变化？

4. 在创新时，哪些元素必须保持以维持用户预设？
   - 哪些细节一旦改变，会让用户察觉"不真实"？

【案例】军人回家视频
1. 用户认为：真实的家庭录像
2. 用户预设：这是真实发生的军人回家
3. 爆款价值：100%依赖于"真实性"
4. 必须保持：UGC手持拍摄感、自然光线、非专业构图、真实的家庭环境
```

### 解决方案5：增加"空间关系"强制检查

在"阶段三"生成提示词前，增加空间分析步骤：

```markdown
【阶段二.5】空间关系强制分析（新增）

在生成镜头内脚本前，必须先完成空间分析：

【强制建立空间坐标系】
1. 镜头位置：摄像机在哪里？（例如：画面左前方）
2. 主要角色位置：
   - 角色A在哪里？（例如：画面中央，距镜头3米）
   - 角色B在哪里？（例如：画面右后方，距镜头8米，距角色A 5米）
   - 角色C在哪里？（例如：镜头后方，正在拍摄）

【强制描述移动路径】
对于任何移动，必须描述：
1. 起点：角色从哪里开始移动
2. 方向：向哪个方向移动（面向镜头、背向镜头、从左到右、从右到左）
3. 距离：移动了多远（用米或步数表示）
4. 终点：最终到达哪里
5. 相对位置变化：与其他角色的距离如何变化

【示例】正确的描述方式：
✅ "婴儿从画面中央的地毯上（距镜头约2米），面向镜头方向，爬行了1.5米，到达镜头正前方（距镜头0.5米），此时母亲在镜头后方拍摄。婴儿随后转身180度，面向画面右后方（此时父亲的位置，距婴儿约4米），再次爬行约4米，到达父亲身边。"

❌ "婴儿爬向母亲，然后又爬向父亲。"（空间信息缺失）

【强制检查清单】
在生成每个分镜的提示词后，检查：
✅ 所有角色的位置是否清晰？
✅ 移动的方向和距离是否明确？
✅ 空间关系是否符合物理逻辑？
✅ 读者能否在脑海中画出场景平面图？
```

## 修改策略总结

### 需要在提示词中增加的内容：

1. **在"第一步"后增加：**
   - 真实性层级判断

2. **在"第二步"扩展：**
   - 增加"用户认知预设分析"
   - 扩展"核心情绪价值"，增加社会性情绪类别

3. **在"维度3"修改：**
   - 重构"爆款因子"识别，分为三层（身份符号>动作>场景）

4. **在"阶段二"和"阶段三"之间增加：**
   - "阶段二.5：空间关系强制分析"

5. **在"输出格式"中增加：**
   - 真实性层级
   - 用户认知预设
   - 身份符号分析
   - 空间关系检查清单

### 关键原则：

1. **社会价值 > 个人情绪**
   - 职业身份、社会角色、文化符号的价值高于普通情绪

2. **真实性是放大器**
   - 对于UGC类视频，真实性比美观度更重要
   - 任何破坏真实性的创新都是灾难

3. **身份符号 > 动作**
   - 一个军装的价值 > 一千个温情动作

4. **空间清晰度 = 可执行性**
   - 如果AI说不清角色在哪里，Sora也生成不出来

5. **用户视角优先**
   - 不是问"视频讲了什么"，而是问"用户以为看到了什么"
